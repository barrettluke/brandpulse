## üöÄ Why BrandPulse?
GitHub READMEs are the front door to your projects. Manually creating banners in image editors takes too long, and generic templates lack personality. **BrandPulse** solves this by providing a terminal-first, high-fidelity branding engine.

### Use Cases:
- **Project Launches**: Professionalize a new repository in under 5 seconds.
- **Unified Ecosystems**: Give all your private/public tools a consistent visual identity.
- **AI-Native Banners**: Use background images generated by Grok or Gemini and let BrandPulse handle the typography and styling.
- **Agent Workflows**: Perfect for AI agents (like Lex) to automatically brand new projects they create for you.

## üõ†Ô∏è Usage

### Installation
Requires Python 3 and the `Pillow` library.
```bash
pip install Pillow
```

### Basic Examples
**1. The Classic Look**
```bash
python3 brand.py "My Project"
```

**2. Modern Tech Style**
```bash
python3 brand.py "LocalOps" --theme ocean --font space --pattern waves --align left
```

**3. Retro Gaming / 8-Bit**
```bash
python3 brand.py "QuestLog" --theme sunset --font press-start --pattern rays --vignette
```

**4. Code-Centric / Matrix**
```bash
python3 brand.py "Mainframe" --theme matrix --pattern circuit --border 10
```

### Advanced: AI Backgrounds
If you have an AI-generated image (e.g., from Gemini), use it as the backdrop:
```bash
# Uses custom image and skip text if the name is already in the art
python3 brand.py "Project Name" --bg background.png --no-text
```

## üî° Available Options
| Flag | Options | Description |
| :--- | :--- | :--- |
| `--theme` | `cyberpunk`, `matrix`, `sunset`, `forest`, `ocean`, `mono` | Color preset |
| `--font` | `orbitron`, `space`, `press-start`, `inter` | Automatic Google Font |
| `--pattern` | `grid`, `dots`, `hex`, `rays`, `waves`, `circuit`, `stars` | BG geometry |
| `--align` | `left`, `center`, `right` | Title position |
| `--glow` | `center`, `sides`, `corners`, `scatter`, `none` | Glow shape position |
| `--random` | (None) | Randomizes all styles |
